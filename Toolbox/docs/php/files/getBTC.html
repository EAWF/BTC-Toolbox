getBTC.php

File containing the Bitcoin Merchants Toolbox functions

See https://github.com/EAWF/Bitcoin-Merchants-Toolbox for installation & usage instructions.

Classes

ExtendedPublicKey Class encapsulating data of an Extended Public Key and the CKDPub function
HashUtil Class containing Hash functions relevant to Bitcoin
Bech32 Class for encoding and decoding Bech32 Bitcoin addresses
Singleton Singleton design pattern implementation
BitcoinECC Class that contains static helper utilities for performing calculations along the SECP256K1 elliptic curve

Derive a Bitcoin Address from a named account at the given index

Main interaction function to derive a RECEIVING (external) address from a given extended public key that is associated with the specified name in the conf file.

Parameters

string $name

Extended public key to reference in the conf file
integer $childNum

Child number index to derive a public key from

Returns

string —

Encoded address derived from the given extended public key at the given child number index

getBTCBalance()

getBTCBalance(string  $address, integer  $confirmations) : float

Checks the balance (in BTC units) of a Bitcoin Address with an optional minimum number of confirmations

Uses the Blockstream Esplora API to retrieve UTXOs of the given address, then checks whether the UTXOs have greater than or an equal amount of confirmations specified.

Parameters

string $address

Bitcoin Address to check the address of
integer $confirmations

(Optional, default = 0) Minimum number of confirmations to consider a UTXO as part of the balance

Returns

float —

Bitcoin amount
